### 分镜图生成

**状态**: 规划中
**优先级**: 高

#### 功能描述

基于分镜脚本的画面描述，自动生成对应的分镜图片。

#### 技术方案

- **API 接口**: 使用阿里云 AI 图片生成 API（如通义万相）
- **生成模式**:
  - 单条生成：为指定分镜生成图片
  - 批量生成：一键为所有分镜生成图片
  - 重新生成：对不满意的图片重新生成
- **版本管理**:
  - 每条分镜支持生成多个版本
  - 支持版本切换（上一版/下一版）
  - 记录生成历史

#### 任务清单

- [ ] 创建/完善 `src/api/imageGeneration.ts`
- [ ] 实现异步任务创建 + 轮询
- [ ] 创建 `src/utils/imageQueue.ts`（并发控制）
- [ ] 更新 `ScriptTable.vue` 增加分镜图状态列
- [ ] 支持当前方案批量生成分镜图
- [ ] 支持单条生成/重试/重生

#### UI 设计

在 `ScriptTable.vue` 中新增分镜图列：
- 展示生成的分镜图缩略图
- 生成按钮（生成分镜图 / 重新生成）
- 版本切换控件（多版本时显示）
- 生成状态指示（生成中 / 成功 / 失败）

#### 依赖条件

- [ ] 阿里云图片生成 API 接入
- [ ] 图片存储方案（OSS）
- [ ] 异步任务队列管理

### 视频生成

**状态**: 规划中
**优先级**: 中

#### 功能描述

基于分镜脚本和分镜图，自动生成视频片段或完整视频。

#### 技术方案

- **API 接口**: 使用阿里云 AI 视频生成 API
- **生成模式**:
  - 单片段生成：为指定分镜生成视频片段
  - 批量生成：一键生成所有分镜的视频
  - 合成完整视频：将所有片段合成为完整视频
- **输入来源**:
  - 分析模式：使用原始视频片段
  - 创作模式：基于分镜图 + 脚本生成

#### 任务清单

- [ ] 创建 `src/api/videoGeneration.ts`
- [ ] 基于"当前选中脚本 + 分镜图"组装请求参数
- [ ] 新增视频生成入口与状态展示（待开始/生成中/已完成/失败）
- [ ] 支持失败重试与结果回填

#### UI 设计

在 `ScriptTable.vue` 中新增视频预览列：
- 展示生成的视频预览
- 生成按钮（生成视频 / 重新生成）
- 版本切换控件
- 播放控制

#### 依赖条件

- [ ] 阿里云视频生成 API 接入
- [ ] 视频存储方案
- [ ] 视频合成能力
- [ ] 分镜图生成功能（前置依赖）

### 移动端适配

**状态**: 规划中
**优先级**: 中

#### 功能描述

支持移动端（手机/平板）访问和使用。

#### 技术方案

- 响应式布局：使用 Tailwind CSS 断点
- 移动端导航：底部 Tab 导航
- 触摸手势：支持滑动、缩放等手势操作
- 性能优化：懒加载、虚拟滚动

#### 适配范围

|------|----------|
| 首页 | 单列布局 |
| 分析页 | 全屏预览 + 抽屉面板 |
| 生成页 | 分步引导式界面 |
| 收藏页 | 卡片列表 |
