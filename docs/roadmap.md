### 移动端适配

**状态**: 规划中
**优先级**: 中

#### 功能描述

支持移动端（手机/平板）访问和使用。

#### 技术方案

- 响应式布局：使用 Tailwind CSS 断点
- 移动端导航：底部 Tab 导航
- 触摸手势：支持滑动、缩放等手势操作
- 性能优化：懒加载、虚拟滚动

#### 适配范围

| 页面 | 适配方案 |
|------|----------|
| 首页 | 单列布局 |
| 分析页 | 全屏预览 + 抽屉面板 |
| 生成页 | 分步引导式界面 |
| 收藏页 | 卡片列表 |
| 个人中心 | 简化表单 |

#### 任务清单

- [ ] 响应式布局适配
- [ ] 移动端导航组件
- [ ] 触摸手势支持
