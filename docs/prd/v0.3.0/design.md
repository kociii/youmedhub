# v0.3.0 界面设计文档

> 无限画布编辑器界面设计，支持脚本、文本、图片、视频节点

## 1. 页面结构概览

### 1.1 页面清单

| 页面 | 路由 | 功能描述 | 优先级 |
|------|------|----------|--------|

## 4. 连线设计

### 4.1 连线锚点

每个节点有 5 个锚点（参考 Constella）：

```
           ● top
           │
  ● left ──┼── ● right
           │
           ● bottom

           ● center (节点中心)
```

**锚点可见性**：
- 默认隐藏
- hover 节点时显示
- 拖拽连线时高亮可连接锚点

### 4.2 连线样式

**贝塞尔曲线（默认）**：
```
  ┌─────┐                ┌─────┐
  │  A  │●──────────────●│  B  │
  └─────┘                └─────┘
```

**直线**：
```
  ┌─────┐              ┌─────┐
  │  A  │──────────────│  B  │
  └─────┘              └─────┘
```

**折线（step）**：
```
  ┌─────┐
  │  A  │●──────┐
  └─────┘       │
                ┌──────●─────┐
                       │  B  │
                       └─────┘
```

### 4.3 箭头样式

| 类型 | 图示 |
|------|------|
| arrow | ▶ |
| diamond | ◆ |
| circle | ● |
| none | 无 |

## 8. 响应式适配

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| 桌面端 | ≥1280px | 三栏：左侧面板(220px) + 画布 + 右侧属性栏(260px) |
| 小桌面 | 1024-1279px | 两栏：隐藏左侧面板，右侧属性栏改为浮动抽屉 |
| 平板端 | 768-1023px | 单栏：画布全屏，底部工具栏简化，属性栏改为底部抽屉 |
| 移动端 | <768px | 单栏 + 底部导航，画布全屏（Phase 4+） |
