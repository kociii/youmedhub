# v0.3.0 界面设计文档

> 项目模块与全屏项目画布（MVP）

---

## 1. 当前实现审查结论

本版设计参考并审查了：

- `src/App.vue`
- `src/router/index.ts`
- `src/components/layout/AppLayout.vue`
- `src/components/layout/AppMenu.vue`
- `src/views/AnalyzePage.vue`
- `src/views/CreatePage.vue`
- `src/components/RightPanel.vue`
- `src/components/ResultToolbar.vue`
- `src/components/ScriptTable.vue`
- `src/composables/useVideoAnalysis.ts`

### 结论

- 当前主应用壳层适合承载“项目管理页”。
- 当前结果工具栏最适合承载“新建项目”入口。
- 项目画布必须独立全屏，否则空间不足。

---

## 2. 页面结构概览

| 页面 | 路由 | 承载方式 | 优先级 |
|------|------|----------|--------|
| 项目管理页 | `/projects` | 继续使用主应用壳层 | P0 |
| 项目画布页 | `/projects/:id` | 新标签页全屏打开 | P0 |

## 6. 节点设计

### 6.1 脚本卡片（`script`）

- 从当前脚本结果导入
- 展示：
  - 分镜序号
  - 景别
  - 运镜
  - 画面内容
  - 口播
  - 时长

### 6.2 便签卡片（`note`）

- 手动新增
- 用于补充拍摄说明、创意备注

---

## 7. 关键交互

### 7.1 从结果进入项目

```
分析页 / 创作页
  └─ 点击「新建项目」
       └─ 创建项目主记录
            └─ 创建项目初始快照
                 └─ 新标签页打开 /projects/:id
```

### 7.2 从项目管理页进入项目

```
/projects
  ├─ 点击「新建项目」 -> 创建空白项目 -> 新标签页打开
  └─ 点击「打开项目」 -> 新标签页打开现有项目
```
