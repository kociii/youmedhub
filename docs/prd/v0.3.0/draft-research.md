# v0.3.x 画布能力调研草稿

> 本文件保留为 0.3.x 系列的共享调研池，不等同于单一版本承诺范围。

## 使用说明

- `v0.3.0` 仅消费其中与“全屏画布打开、脚本导入、基础交互”直接相关的部分。
- `v0.3.1` 再吸收属性编辑、更多节点类型、项目管理相关方案。
- `v0.3.2` 再评估连线、公开分享、导出与项目内 AI。

## 当前拆分原则

1. 先解决 Web 应用中“新标签页打开全屏画布”的基础链路。
2. 再补编辑增强。
3. 最后补连接、云端和 AI 生成能力。
