## 当前实现审查结论

参考文件：

- `src/router/index.ts`
- `src/components/layout/AppLayout.vue`
- `src/components/ResultToolbar.vue`
- `src/composables/useVideoAnalysis.ts`
- `src/views/ProjectCanvasPage.vue`
- `src/projects/types.ts`

### 结论

- 项目画布页继续保持新标签页全屏体验。
- AI 触发入口更适合放在项目节点卡片和属性栏中，而不是主应用结果页。
- 当前项目画布仍是 DOM 卡片式编辑器，连线图层应叠加在 DOM 节点层上。

## 页面与交互增量

### 编辑页新增

- 节点锚点
- 连线图层
- 云端保存状态
- 导出菜单
- 节点 AI 操作按钮
- 公开分享按钮
- 主题切换按钮

### `ai_call` 卡片布局（新增）

```
┌──────────────────────────────────────────┐
│ AI 调用卡片                      [运行]  │
├──────────────────────────────────────────┤
│ 模型: qwen-xxx                           │
│ 输出类型: markdown                        │
│ 输入绑定: text_node_1, script_node_2      │
├──────────────────────────────────────────┤
│ 输出区（固定在卡片下半区）                │
│ - running: 显示流式内容/进度              │
│ - success: 按 output_type 渲染            │
│ - error: 显示错误信息与重试按钮            │
└──────────────────────────────────────────┘
```

输出渲染规则：

1. `text/markdown/json`：文本渲染区域
2. `image`：图片预览区域（来源 OSS）
3. `video`：视频预览区域（来源 OSS）
4. `script`：脚本结构化预览（可转为 script 节点）

### 公开分享页新增

- 路由建议：`/share/projects/:token`
- 以独立全屏只读页打开
- 默认不显示主应用菜单
- 页面内容：
  - 项目标题
  - 作者昵称（可选）
  - 分享说明
  - 只读项目画布
  - “复制此项目”按钮（可选，后续）

### 项目列表页新增

- 本地 / 云端项目切换
- 最近更新时间
- 同步状态

## 设计原则

- 连线与 AI 不应破坏前两版已经形成的“全屏专注编辑”体验。
- 云端保存状态尽量轻量化展示，避免顶部工具栏过载。
- 公开分享默认只读，避免和编辑态权限混淆。

## 节点交互规则补充

- 画布内节点添加入口固定在左侧。
- 节点显示顺序统一为：
  - 文本
  - 图片
  - 视频
  - 脚本
  - AI 节点
  - 便签
- `note` 节点不支持连线。
- `note` 节点仍支持：
  - 文本修改
  - 拖拽移动
  - 边缘缩放

## 主题模式

- 画布支持浅色模式和暗黑模式。
- 主题切换至少影响：
  - 背景色
  - 网格线
  - 节点容器
  - 连线颜色
  - 浮层 / 属性栏背景
