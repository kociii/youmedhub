# v0.2.4 界面设计文档

> 存储方案验证与收尾 - 界面结构设计与交互优化

---

## 1. 页面结构概览

### 1.1 页面清单

| 页面 | 路由 | 功能描述 | 优先级 |
|------|------|----------|--------|
| 视频分析页 | /analyze | 上传视频并分析脚本结构 | P0 |
| 脚本生成页 | /create | 从零创作或参考生成脚本 | P0 |
| 收藏列表页 | /favorites | 查看和管理收藏的脚本 | P1 |
| 个人中心页 | /profile | 用户资料和设置 | P1 |

### 1.2 导航结构

```
┌─────────────────────────────────────────────────────────┐
│  Logo      │  拆解脚本  │  脚本生成  │  我的收藏  │  👤  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│                    主内容区域                           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 2. 页面详细设计

### 2.1 视频分析页 (/analyze)

**功能描述**：
用户上传本地视频文件，AI 分析视频内容并生成分镜脚本。

**页面布局**：

```
┌─────────────────────────────────────────────────────────┐
│  拆解脚本                                     [用户头像] │
├─────────────────────────────────────────────────────────┤
│  ┌─────────────────────┐  ┌─────────────────────────┐  │
│  │  模型配置            │  │                         │  │
│  │  ─────────           │  │     视频预览区域         │  │
│  │  ○ qwen3.5-flash    │  │                         │  │
│  │  ○ qwen3.5-plus     │  │   [点击上传或拖拽视频]   │  │
│  │                      │  │                         │  │
│  │  ☑ 启用思考模式      │  │   ┌─────────────────┐   │  │
│  │  ─────────           │  │   │                 │   │  │
│  │  上传设置            │  │   │   视频播放器    │   │  │
│  │  [上传进度条]        │  │   │                 │   │  │
│  │  上传中... 45%       │  │   │                 │   │  │
│  │                      │  │   └─────────────────┘   │  │
│  │                      │  │                         │  │
│  │                      │  │   [开始分析]            │  │
│  │                      │  │                         │  │
│  └─────────────────────┘  └─────────────────────────┘  │
│                                                         │
│  ┌───────────────────────────────────────────────────┐ │
│  │  结果展示区域                                      │ │
│  │  ──────────────────────────────────────────────── │ │
│  │  [Markdown] [分镜表格]                            │ │
│  │  ┌─────────┬────────┬──────────┬────────┬────────┐│ │
│  │  │ 时间    │ 景别   │ 画面描述  │ 台词   │ 时长   ││ │
│  │  ├─────────┼────────┼──────────┼────────┼────────┤│ │
│  │  │ 00:00   │ 特写   │ ...      │ ...    │ 3s     ││ │
│  │  └─────────┴────────┴──────────┴────────┴────────┘│ │
│  └───────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

**交互说明**：
1. 选择视频文件 → 生成本地预览（blob URL）→ 显示上传进度 → 完成后显示 "上传完成"
2. 点击 [开始分析] → 显示分析中状态 → 流式输出 Markdown 结果
3. 切换 [Markdown]/[分镜表格] Tab → 切换展示形式
4. 模型切换 → 清空已上传状态，提示重新上传

### 2.2 脚本生成页 (/create)

**功能描述**：
支持两种模式：从零创作和参考生成。

**Tab 切换布局**：

```
┌─────────────────────────────────────────────────────────┐
│  脚本生成                                     [用户头像] │
├─────────────────────────────────────────────────────────┤
│  [ 从零创作 ▼ ]  [ 参考生成 ▼ ]                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ╔═══════════════════════════════════════════════════╗ │
│  ║  从零创作模式                                      ║ │
│  ╠═══════════════════════════════════════════════════╣ │
│  ║  上传参考视频（可选）                                ║ │
│  ║  ┌─────────────────────────────────────────────┐  ║ │
│  ║  │ [点击上传视频] 或 [拖拽到此处]              │  ║ │
│  ║  └─────────────────────────────────────────────┘  ║ │
│  ║                                                   ║ │
│  ║  创作需求                                         ║ │
│  ║  ┌─────────────────────────────────────────────┐  ║ │
│  ║  │ 描述你想要的脚本风格、目标受众、时长等...    │  ║ │
│  ║  │                                             │  ║ │
│  ║  └─────────────────────────────────────────────┘  ║ │
│  ║                                                   ║ │
│  ║  [ 生成脚本 ]                                     ║ │
│  ╚═══════════════════════════════════════════════════╝ │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**参考生成模式布局**：

```
┌─────────────────────────────────────────────────────────┐
│  ╔═══════════════════════════════════════════════════╗ │
│  ║  参考生成模式                                      ║ │
│  ╠═══════════════════════════════════════════════════╣ │
│  ║  参考视频（可选）                                  ║ │
│  ║  ┌─────────────────────────────────────────────┐  ║ │
│  ║  │ [点击上传视频]                              │  ║ │
│  ║  └─────────────────────────────────────────────┘  ║ │
│  ║                                                   ║ │
│  ║  参考图片（可选）                                  ║ │
│  ║  ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌──────────────┐  ║ │
│  ║  │ +  │ │ 🖼️ │ │ 🖼️ │ │ 🖼️ │ │ [+ 添加图片] │  ║ │
│  ║  └────┘ └────┘ └────┘ └────┘ └──────────────┘  ║ │
│  ║                                                   ║ │
│  ║  参考脚本                                          ║ │
│  ║  ┌─────────────────────────────────────────────┐  ║ │
│  ║  │ 粘贴参考的脚本内容...                        │  ║ │
│  ║  │                                             │  ║ │
│  ║  └─────────────────────────────────────────────┘  ║ │
│  ║                                                   ║ │
│  ║  [ 生成脚本 ]                                     ║ │
│  ╚═══════════════════════════════════════════════════╝ │
└─────────────────────────────────────────────────────────┘
```

**交互说明**：
1. Tab 切换 → 切换面板内容，保留当前输入
2. 图片上传 → 支持多选，本地预览，提交时上传
3. 点击 [生成脚本] → 如有文件先上传 → 调用 AI → 显示结果
4. 结果可收藏、导出 Excel

## 4. 响应式适配

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| 桌面端 | ≥1280px | 双栏布局（左配置 + 右内容） |
| 平板端 | 768-1279px | 单栏布局，配置面板折叠到顶部 |
| 移动端 | <768px | 底部导航栏，配置面板全屏弹窗 |

**移动端底部导航**：

```
┌─────────────────────────────────┐
│                                 │

├─────────────────────────────────┤

└─────────────────────────────────┘
```
