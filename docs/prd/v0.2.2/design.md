### 3.2 脚本生成页面（统一输入 + 三步流程）

> 不再区分「从零创作 / 参考生成」Tab。  
> 用户在同一表单中输入要素，系统按输入内容自动判定策略（是否使用参考脚本）。
> 将【创作主题】升级为【视频要求】卡片，采用「类型选项 + 动态输入框 + 补充说明」交互。

### 3.2.1 视频类型与动态字段规范

> 每次仅选择 1 个视频类型；下方自动加载对应输入框。  
> 「补充说明」在所有类型下都保留，用于自由补充业务约束。

| 视频类型 | 动态输入字段（示例） |
|----------|----------------------|
| 电商头图类 | 商品名称、核心卖点、价格/优惠信息、目标人群 |
| 使用场景类 | 典型场景、用户痛点、使用步骤、场景道具/环境限制 |
| 卖点展示类 | 卖点 1/2/3、证明方式（数据/对比）、优先级 |
| 开箱测评类 | 开箱亮点、测评维度、优缺点、结论倾向 |
| 对比种草类 | 对比对象、对比维度、推荐理由、适用人群 |
| 教程讲解类 | 教程目标、步骤拆解、注意事项、成果展示方式 |
| 品牌故事类 | 品牌定位、故事主线、情感关键词、行动号召 |
