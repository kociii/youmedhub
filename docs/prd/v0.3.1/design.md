## 当前实现审查结论

参考文件：

- `src/App.vue`
- `src/router/index.ts`
- `src/components/layout/AppLayout.vue`
- `src/components/layout/AppMenu.vue`
- `src/components/ResultToolbar.vue`

### 结论

- 主站依旧维持三栏壳层。
- 全屏项目编辑器仍应保持独立标签页体验。
- 项目列表页已在 v0.3.0 建立，本版重点补强编辑器本身。

## 编辑页新增布局

```
┌────────────────────────────────────────────────────────────────────┐
│ 顶部浮层栏：标题 / 撤销 / 重做 / 保存状态                          │
├────────────────────────────────────────────────────────────────────┤
│ 左侧轻量目录  │                画布区域               │ 右侧属性栏 │
│ 节点列表/搜索 │                                     │ 类型/位置   │
│               │                                     │ 尺寸/内容   │
└────────────────────────────────────────────────────────────────────┘
```

### 说明

- 左侧目录与右侧属性栏均为可折叠浮层，不再回到主应用固定三栏。
- 这样仍满足“全屏展示画布”的主诉求，只是在全屏内加入可收起的编辑辅助区。
